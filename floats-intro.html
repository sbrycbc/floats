Floats

Floats sorgen dafür, das wir Elemente links oder rechts platzieren können.

Ein Elemen dem ein float in css zugewiesen wurde, ist kein block-element mehr, nimmt also keinen reservierten platz mehr auf unserer seite ein.

Float optionen ul>
  • float: left; = platziert das element auf der linken seite
  • float: right; = platziert das element auf der rechten seite
  • float: none; = default, keine platzierung

    • mit float: left; platzieren wir elemente auf der linken seite (grüne box)

    mit float: right; platzieren wir elemente auf der rechten seite (rote box)







    Bilder können in Text, mit Float fließend platziert werden
